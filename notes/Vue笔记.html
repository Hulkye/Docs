# 数据和方法

响应式数据属性

  • Vue实例创建时,data中存在的属性才是响应式的, 实例创建之后再去追加属性不会触发更新
  • 使用 Object.freeze()，这会阻止修改现有的属性, 响应系统无法再追踪变化
    • Object.freeze() 方法可以冻结一个对象，冻结指的是不能向这个对象添加新的属性，不能修改其已有属性的值，不能删除已有属性，以及不能修改该对象已有属性的可枚举性、可配置性、可写性。该方法返回被冻结的对象。详情查看 MDN

实例属性与方法

  • Vue 实例暴露了许多有用的属性和方法,他们都有前缀 $ , 来区分用户定义的属性

egg:

// 上面Vue实例中的    el:"#app"   等同于
vm.$el === document.getElementById('app')

// 上面Vue实例中的    data: data   等同于
vm.$data === data

// $watch 是一个实例方法
vm.$watch('a', function (newValue, oldValue) {
  // 这个回调将在 `vm.a` 改变后调用

# 生命周期钩子

每一个Vue实例在创建初始化过程中, 都会 运行生命周期钩子的函数

  • 生命周期钩子中的this指向调用它的Vue实例
  • 不要在选项属性或回调上使用箭头函数

钩子函数

  • beforeCreate > 实例初始化后,数据观测和event/watcher 事件配置之前被调用
  • created > 在实例创建完成后被立即调用 (挂载阶段还没开始)
  • beforeMount > 在挂载开始之前被调用：相关的 render 函数首次被调用。
  • mounted > 可以理解为组件挂载后被调用
  • beforeUpdate > 数据更新时调用, 发生在虚拟 DOM 打补丁之前
  • updated > 由于数据更改导致的虚拟 DOM 重新渲染和打补丁，在这之后会调用该钩子
  • activated > keep-alive 组件激活时调用
  • deactivated > keep-alive 组件停用时调用
  • beforeDestroy > keep-alive 组件停用时调用
  • destroyed > Vue 实例销毁后调用

注意: mounted 与 updated 不会等所有子组件挂载/更新后才调用, 可以用 vm.$nextTick 处理

# 常用指令

# 插值表达式

作用: 将data中的数据渲染到页面上

使用: 在标签的innerHTML位置写上

合法用法:

  • 变量名
  • 字符串拼接
  • 数值运算
  • 三元表达式
  • 函数

# v-text

**作用：**将data中的数据渲染到页面上

使用方法：在标签的属性位置写上 v-text="data中的变量"

合法用法：

  1. 变量名
  2. 字符串拼接
  3. 数值运算
  4. 三元表达式
  5. 函数

# v-html

作用：渲染带特殊html标签的字符串的，如果没有特殊标签也能渲染出来，但是没有v-html的必要了

使用方法：在标签的属性位置写上v-html="data中的变量"

# v-bind

**作用：**动态绑定属性（注意属性可以是任意的属性）

使用方法： 标签的属性位置 :属性名=“data中的变量”

动态绑定类名需要注意，有两种写法。

  1. 直接使用对象形式绑定单个类名：<div :class="{active: isActive}">hahaha</div> 其中isActive已定义在data中
  2. 使用数组形式绑定多个类名：<div :class="[{active: isActive}, {bkblue: isActive}]">heheh</div>其中isActive已定义在data中

# v-model

**作用：**实现双向数据绑定，注意：它只能给input/textarea/select/组件 这几个元素使用

**双向数据绑定：**数据模型和视图模型中的数据同步变化

<input type="text" v-model="msg">msg已在data中定义

# v-for

作用：循环渲染数组或对象

**写法：**在标签属性位置写上 v-for="(item, index) in arr"

**注意：**以下两种情况不能触发视图模型的更新

  1. 通过数组的length属性去改变数组时不能触发视图更新
  2. 通过数组的索引去改变数组时不能触发视图更新

解决方法：

  1. 通过数组的splice方法解决
  2. 通过Vue.set(需要改变的数组，数组项的索引，新的值)： 例如Vue.set(vm.users, 0, {name: 'rose', id: 3234})

**注意：**使用v-for指令必须结合key属性使用，key属性作用是唯一标识数组项，提升渲染性能用的

# v-on

**作用：**监听dom事件

写法：在标签属性位置写上 @事件类型=“methods中定义的函数” （其中事件类型可以是自定义的）

**传递事件对象：**使用$event 注意不能叫引号

**事件修饰符：**增强事件用的，常用的.prevent .stop

**按键修饰符：**针对键盘事件来说，常用的.enter

# v-show和v-if

**作用：**用来控制元素的显示和隐藏

**区别：**v-show通过样式display:none；v-if通过直接操作dom

**使用场景区别：**大量dom频繁切换的时候用v-show合适；异步数据渲染用v-if合适
